Octopus

self-hosted. one command. docker required.

Install on Linux / macOS

curl -fsSL https://mk-amorson.github.io/Octopus/install | sh

Install on Windows (PowerShell)

iwr -useb https://mk-amorson.github.io/Octopus/install.ps1 | iex

What it does

Downloads the octopus CLI, adds it to your PATH, and runs octopus install. The installer asks where to bind (localhost or all interfaces), which subpath to serve under (default /octopus), and which host port to use, then builds and starts the app under Docker.

Afterwards

octopus start
octopus stop
octopus status
octopus update
octopus uninstall
